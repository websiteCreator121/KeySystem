# 🔑 KeySystem

A simple activation system for generating lifetime keys.

## ✨ Features
- 🔐 Generate lifetime activation keys
- ⚡ Fast and lightweight
- 🛠️ Easy to use
- 🎨 Clean and modern interface

## 🩹 SystemHelper.exe
If your generated key doesn't work, run **SystemHelper.exe**.

It will automatically try multiple repair methods to fix common issues with the key generator and activation process.

> **Note:** Only use **SystemHelper.exe** if your generated key fails to activate.

## 👤 Touch Fling
When another player or person touches your character, they will be **flung automatically**.
